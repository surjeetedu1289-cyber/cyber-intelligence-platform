# File Guide

This document explains the purpose of the main files created for the project foundation.

- [README.md](README.md): Project overview, setup steps, and usage instructions.
- [requirements.txt](../requirements.txt): Python dependencies for the FastAPI backend.
- [package.json](../package.json): Node.js scripts and frontend dependencies.
- [.gitignore](../.gitignore): Ignores local environment files, build artifacts, and Python cache output.
- [backend/__init__.py](../backend/__init__.py): Marks the backend package for Python imports.
- [backend/collectors/__init__.py](../backend/collectors/__init__.py): Marks the collectors package.
- [backend/models/__init__.py](../backend/models/__init__.py): Marks the models package.
- [backend/services/__init__.py](../backend/services/__init__.py): Marks the services package.
- [config](../config): Application configuration directory for environment-specific settings.
- [data](../data): Persistent storage location for collected intelligence assets.
- [docs](../docs): Documentation and architecture notes.
- [frontend](../frontend): React + TypeScript + TailwindCSS user interface.
- [tests](../tests): Automated tests for the platform.
- [.github/workflows](../.github/workflows): CI/CD automation.
